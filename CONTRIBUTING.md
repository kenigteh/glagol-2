# Контрибьютинг

Спасибо за интерес к Glagol! Любой вклад приветствуется — баг-репорт, идея, правка кода или документации.

## 🐛 Баги и 💡 идеи

Открой [issue](https://github.com/kenigteh/glagol-2/issues) по соответствующему шаблону. Для бага укажи версию macOS, чип, версию Glagol и шаги воспроизведения.

## 🔧 Pull requests

1. Форкни репозиторий и создай ветку: `feat/...` для фич, `fix/...` для багфиксов.
2. Собери проект — см. [Сборка из исходников](README.md#сборка-из-исходников).
3. Держи PR сфокусированным: **один PR — одна задача**.
4. Опиши, **что** меняешь и **зачем**.

## 📐 Стиль кода

- **Swift 6**, macOS 15+, Apple Silicon.
- Осмысленные имена, ранние возвраты, без лишних зависимостей.
- Архитектура batch (запись копится до stop'а, затем разом в ASR) — см. [README](README.md#архитектура).
- Перед PR убедись, что проект **собирается** и приложение **запускается**, а диктовка работает.

Вопросы — через [issue](https://github.com/kenigteh/glagol-2/issues). Спасибо ❤️
